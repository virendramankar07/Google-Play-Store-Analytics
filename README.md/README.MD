# 📊 Google Play Store Analytics Dashboard

## 🚀 Project Overview

This project analyzes Google Play Store application data to uncover insights related to app ratings, installs, reviews, pricing, revenue potential, and category trends using Python, SQL, and Power BI.

---

# 🛠 Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* SQL
* Power BI
* DAX
* Machine Learning

---

# 📂 Project Workflow

## 1️⃣ Data Cleaning

* Removed duplicates
* Handled missing values
* Converted data types
* Cleaned installs & price columns

## 2️⃣ Exploratory Data Analysis (EDA)

* Rating Distribution
* Top Categories
* Installs Analysis
* Reviews vs Ratings
* Paid vs Free Apps

## 3️⃣ SQL Analysis

* Top Installed Apps
* Revenue Analysis
* Highest Rated Categories
* Reviews Analysis

## 4️⃣ Power BI Dashboard

* KPI Cards
* Interactive Filters
* Revenue Insights
* Sentiment Analysis
* Top Apps Analysis

# 📈 Dashboard Features

✅ Total Apps KPI

✅ Revenue Analysis
✅ Reviews vs Ratings
✅ Category Analysis
✅ Dynamic Filters
✅ Interactive Visualizations

---

# 💡 Key Insights

* Most apps on Play Store are Free Apps
* Family category dominates app count
* Higher reviews generally lead to higher installs
* Business apps show strong revenue potential
